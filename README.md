## このリポジトリの構造

生物学的神経系にインスパイアされた「自己進化型AI管理OS」構造を採用。
詳細は `docs/ARCHITECTURE.md` を参照。

```
/
├── CLAUDE.md              # AI指令の核心（大脳皮質）
├── .clauderules           # 自動行動規則（本能）
├── docs/                  # 設計書・操作マニュアル（長期記憶）
├── plans/                 # 作業計画（前頭前皮質）
├── memories/              # 経験・失敗の蓄積（海馬）
├── scratchpad/            # 一時作業領域（作業記憶）
├── core/                  # 基盤コード（脳幹）
├── src/                   # フロントエンド（運動野）
├── api/                   # バックエンドAPI（感覚処理野）
├── tests/                 # テスト
└── infra/                 # インフラ設定
```
